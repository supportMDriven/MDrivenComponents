
# Store Complaint Handling App
The store complaint handling app is a model for a clothing shop, to help staff handle customer complaints professionally..


## Usage
1. Merge the Store Complaint Handling App Model into your model using TKLiveView.
2. User interface messages for all states
3. Two users
    - john@acme.inc (store employee)
    - admin@acme.inc (system manager)
    - The password is "123456" for both users
    - 200 fake complaints in various states of handling
4. MDrivenServer log-on
    - User: "a"
    - Password: "123456" <br><br>
